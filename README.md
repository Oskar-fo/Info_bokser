# Info_bokser
Prosjektet som jeg tenker å lage skal være en egen designa nettside som jeg skal lage til å vise meg ntyttig informasjon jeg kan trenge.
For hvordan den skal se ut kommer jeg til å ta litt inspriasjon fra bildet som er lagt til for denne oppgaven på teams.

Jeg tror jeg kommer til å bruke html, py, sql og css som språk og skolen pc-en min og raspberrypi til utstyr.

Min kompetanse i fagene er dirft:kan sette opp og vedlikeholde server og nettverk.
Utvikling: kan skive noen basice program i de forskjellige språkene. 
Brukerstøtte: kan hjelpe folk med problemer de har mad pc siden jeg var tek supporten til ungdomsskole klassen min
